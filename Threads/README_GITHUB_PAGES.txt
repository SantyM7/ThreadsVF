Proyecto convertido a HTML estático para GitHub Pages.

Cambios realizados:
- Archivos .php convertidos a .html
- Eliminadas dependencias de MySQL/PHP
- Navegación adaptada para HTML estático

IMPORTANTE:
El formulario de contacto ya no usa PHP.
Para hacerlo funcionar en GitHub Pages puedes usar:
- Formspree: https://formspree.io/
- EmailJS: https://www.emailjs.com/

Actualmente el formulario apunta a:
https://formspree.io/f/your-form-id

Debes reemplazar "your-form-id" con tu ID real.

Cómo subir a GitHub:
1. Crea un repositorio nuevo
2. Sube el contenido de la carpeta Threads
3. Ve a Settings > Pages
4. Activa GitHub Pages usando la rama main
5. Tu sitio quedará publicado
